**Salary Prediction & AI Insight Dashboard**

An end-to-end application that predicts data science salaries and explains the results using AI and interactive visualizations.

---
What It Does

- Takes job inputs (experience, role, company size, location, etc.)
- Predicts salary using a trained ML model
- Generates a short AI insight (via Ollama)
- Displays results in an interactive Streamlit dashboard
- Stores predictions in Supabase

---
Architecture

- **Model**: Regression model (Random Forest)
- **Backend**: FastAPI
- **Frontend**: Streamlit
- **AI Insight**: Ollama (qwen3.5:4b)
- **Database**: Supabase

---
Dashboard Features

- KPI-based prediction summary
- Salary benchmark & career-level visuals
- AI-generated commentary
- Comparison with recent predictions
